Detailed Description

This class is used to store and acess multiple Idioms.

This class contains a map what will allow us to store and acess our Idioms in an efficient way. If we try to acess an Idiom that does not exist, the class will handle that case.

◆ decode()

void Idiom_Set::decode ( )

Prints to the standard output channel the decoded version of a String.

Precondition
There are two Strings Name, Text waiting in the standard input channel
Postcondition
If the Idiom Name is in the set and the String Text can be decoded in this idiom, the decoded version of the string is printed to the standard output channel. Otherwise, an error message is printed. O( N ), where N is the length of Text

◆ write_codes()

void Idiom_Set::write_codes ( ) const

Writes the code(s) of an Idiom to the standard output channel.

Precondition
There is a String Name and a String Code waiting in the standard input channel
Postcondition
If the Idiom Name is in the set: if Code equals "todos" the codes of Idiom Name have been written to the standard output channel. if Code equals one of the characters of Idiom Name, the code for that character has been written to the standar output channel. Otherwise, an error message is printed. O( M*log(N) ), where N is the number of characters in the idiom and M is the number of codes that we want to print (M = N if Code = "todos". M = 1 otherwise).
